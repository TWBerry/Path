# Bash Dungeon

A retro ASCII roguelike written entirely in **Bash**.
No external dependencies. Pure terminal magic.

## Features
- Procedurally generated maps
- ANSI color UI + box drawing
- NPC & trading system (buy / sell)
- Inventory + equipment slots
- Player stats & combat
- Random name generation for NPCs

## How to play

Clone and run:

```bash
git clone https://github.com/TWBerry/Path.git
cd Path
./game.sh
